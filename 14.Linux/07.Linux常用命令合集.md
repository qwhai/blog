## Linux常用命令合集

### 1. df

```shell
df -h
df -i
```

### 2. ifconfig

```shell
ifconfig
```

### 3. cat /etc/redhat-release

查看当前系统的redhat版本
```shell
cat /etc/redhat-release
```

### 4. ps -ef

查看运行的进程信息
```shell
ps -ef | grep java
```
